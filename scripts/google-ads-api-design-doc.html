Date: April 2026

Business Model

Esdeveniments.cat is a Catalan cultural events discovery website built with Next.js. The website aggregates and displays local events across Catalonia, helping users find activities in their area. The site is monetized through Google AdSense display advertising.

We want to use the Google Ads API exclusively for Keyword Planner access to:

  • Research monthly search volumes for event-related keywords in Catalan, Spanish, and English
  • Identify trending search terms related to local events and activities
  • Optimize our content and SEO strategy based on actual search demand data

We do not run Google Ads campaigns. Our sole use of the API is the KeywordPlanService for SEO research purposes.

Tool Access/Use

The tool is an internal-only command-line Python script used exclusively by the site owner/developer (Albert Olivé) for periodic keyword research. It is not externally accessible and has no web interface available to the public.

  • Users: 1 internal user (site owner/developer)
  • Access method: Command-line script run locally on developer machine
  • Frequency: Run manually, approximately 1-2 times per week
  • No external users will have access to the tool or its output data

Tool Design

The tool is a Python script (scripts/keyword-planner.py) that:

  1. Accepts seed keywords as command-line arguments (e.g., "que fer avui", "agenda barcelona")
  2. Calls the Google Ads API KeywordPlanIdeaService to retrieve keyword suggestions, monthly search volumes, and competition levels
  3. Outputs results as a Markdown report saved locally to data/seo-audits/
  4. Optionally outputs JSON for further analysis

No data is stored in any external database. Results are saved as local files only.

No campaign management or ad creation is performed.

API Services Called:

  • KeywordPlanIdeaService — Generate keyword ideas with search volume metrics (GenerateKeywordIdeas)
  • KeywordPlanIdeaService — Get historical search volume data (GenerateKeywordHistoricalMetrics)

API Services NOT Used:

  • No CampaignService, AdGroupService, or any campaign management
  • No BiddingStrategyService or budget management
  • No ConversionTrackingService or remarketing
  • No account creation or management for third parties

Data Handling & Privacy

  • No user data is collected or processed — the tool only queries aggregate keyword metrics
  • No Google Ads account data from other companies is accessed
  • API credentials (OAuth2 tokens) are stored as environment variables, never committed to source control
  • The tool accesses only the MCC account owned by the developer (467-859-2752)
  • Output reports are stored locally and not shared publicly

Rate Limiting & Compliance

  • The tool is used manually 1-2 times per week, generating minimal API traffic
  • Each run makes approximately 1-5 API calls (one per seed keyword batch)
  • Built-in rate limiting with delays between requests
  • Full compliance with Google Ads API Terms of Service and Required Minimum Functionality policies
